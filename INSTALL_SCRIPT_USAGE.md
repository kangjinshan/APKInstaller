# APK安装脚本使用说明

这是一个Python脚本，用于将本地APK文件发送到已打开APK安装工具的小米盒子。

## 前置条件

1. ✅ 小米盒子已安装APK安装工具
2. ✅ 小米盒子已打开APK安装工具应用
3. ✅ 在小米盒子上点击"启动接收"按钮
4. ✅ 电脑和小米盒子在同一局域网

## 使用方法

### 基础用法

```bash
# 方式1：使用默认IP（192.168.31.183
python3 install_apk_to_tv.py /path/to/your/app.apk

# 方式2：指定自定义IP地址
python3 install_apk_to_tv.py /path/to/your/app.apk 192.168.1.100
```

### 实际示例

```bash
# 示例1：安装桌面上的APK
python3 install_apk_to_tv.py ~/Desktop/my-app.apk

# 示例2：安装当前目录的debug版本
python3 install_apk_to_tv.py ./app-debug.apk

# 示例3：安装指定路径的APK到自定义IP
python3 install_apk_to_tv.py /Users/kanayama/Downloads/tiktok.apk 192.168.1.200
```

### 修改默认IP地址

如果需要修改默认IP地址，编辑脚本的第10行：

```python
TV_IP = "192.168.31.200"  # 改为你的小米盒子IP
```

## 使用流程

1. **在小米盒子上操作**：
   - 打开"APK安装工具"应用
   - 点击"启动接收"按钮
   - 等待提示"接收服务已启动，端口: 8888"

2. **在电脑上操作**：
   ```bash
   cd AI/APKInstaller
   python3 install_apk_to_tv.py <你的APK文件路径>
   ```

3. **等待传输完成**：
   - 脚本会显示传输进度
   - 传输完成后，小米盒子会自动弹出安装界面
   - 在小米盒子上点击"安装"即可

## 常见问题

### 1. 连接被拒绝
**错误**: `ConnectionRefusedError`

**解决方法**:
- 确保小米盒子已打开APK安装工具
- 确保已点击"启动接收"按钮
- 检查IP地址是否正确

### 2. 连接超时
**错误**: `socket.timeout`

**解决方法**:
- 检查电脑和小米盒子是否在同一局域网
- 确认小米盒子的IP地址
- 检查网络连接是否正常

### 3. 文件不存在
**错误**: `❌ 错误: 文件不存在`

**解决方法**:
- 检查APK文件路径是否正确
- 使用绝对路径或相对路径
- 确保文件名和扩展名正确

## 技术细节

- **传输协议**: TCP Socket
- **端口**: 8888
- **缓冲区大小**: 8KB
- **超时时间**: 30秒
- **支持文件**: 任何大小的APK文件

## 获取小米盒子IP地址

在小米盒子上：
1. 打开"设置"
2. 进入"网络设置"
3. 查看"IP地址"

或者在电脑上使用ADB：
```bash
adb devices -l
```
